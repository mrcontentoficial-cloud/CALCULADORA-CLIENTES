# Cotizador para Clientes · Mr Content

Herramienta self-service para que los clientes armen su plan y soliciten cotizacion.

- Planes prearmados (Esencial, Profesional, Autoridad, Full) + armado a medida.
- Resumen con pago unico y mensualidad por separado.
- Validez de 15 dias naturales.
- Las solicitudes llegan por correo a mrcontentoficial@gmail.com (via FormSubmit).

Abrir `index.html`. Para que lleguen los correos, activar FormSubmit la primera vez (confirmar el correo de activacion).
